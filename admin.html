⚙️ Layout Settings

Everything here lives only in your browser. Nothing is sent anywhere, there is no login, and changes apply to the main page instantly. Come back on the same device and it will still be set up your way. Themes live on the dock; this page shapes everything around them.

← Back to the site

Wondering where the old admin panel went? It is retired on purpose. That panel put content changes behind credentials, and this site is meant to be open. So the power got split the right way: this page hands the layout to you, the visitor, right in your own browser, while the site itself changes out in the open through its public GitHub Pages repo, where I can improve it from anywhere and anyone can read every line.
