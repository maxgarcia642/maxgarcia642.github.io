There is no password screen here, on purpose. A client-side login on a public static site is theater — anyone can read this page's source. The real gate is the fine-grained GitHub token below: without it, nothing on this page can write anything. The token lives only in this tab's memory — never in the repo, never in localStorage — and every write is throttled under GitHub's 80-content-writes/minute secondary limit. Full reasoning in SECURITY.md.
