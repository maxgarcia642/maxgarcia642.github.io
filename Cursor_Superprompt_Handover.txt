ARCHITECTURE CONTRACT (do not break these without an explicit decision)
1. STATIC ONLY. No node server, no framework, no bundler. index.html +
   styles.css + themes.css + script.js + content-loader.js is the whole app.
2. TOKENS ARE THE THEME SYSTEM. themes.css defines 10 blocks keyed on
   [data-theme="..."] (aero glass brutal crt pixel aurora clay neon swiss
   luxe). styles.css contains STRUCTURE ONLY and consumes tokens
   (--bg, --card-bg, --accent, --font-display, --radius-*, etc.).
   A new theme = a new token block + a registry entry in content.json
   ("themes" array) — never new structural CSS per theme.
   FOUC guard: the inline <script> in <head> applies localStorage("mg-theme")
   before first paint. Keep it inline and tiny.
3. DATA LAYER. content.json = site text, projects, arcade entries, theme
   registry. finance.json = Financial Liberty works, market pulse, research
   desk. Renderers live in content-loader.js. To add a finance work or an
   arcade card, edit JSON — not HTML.
4. PERFORMANCE RULES. Animate transform/opacity only. Heavy things load on
   demand: PDF pages render via IntersectionObserver, arcade iframes inject on
   click (facade pattern), TradingView loads on button press, the bubble
   canvas runs only on aero/glass/clay + desktop + no prefers-reduced-motion.
   Respect prefers-reduced-motion everywhere.
5. SECURITY INVARIANTS (SECURITY.md is the law):
   - No secrets in the repo, ever. No tokens, no password hashes.
   - No client-side "login" pretending to be security.
   - Admin auth = fine-grained GitHub PAT, pasted per session, held in a JS
     closure only. Never persist it.
   - Keep SRI on CDN scripts (recompute the sha384 if bumping PDF.js) and keep
     the meta-CSP origins list current when adding embeds/APIs.
   - Admin writes stay throttled under 80 content-writes/minute (currently
     ~1 per 1.2s). Rename = delete+create (no native endpoint).
6. HONESTY FLAGS. Market figures in finance.json are dated (asOf field).
   Contested claims carry a "flags" entry rendered as ⚑ callouts. Never strip
   a flag to make copy prettier; update figures WITH their verification date.
